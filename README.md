# Analise-de-vendas
Projeto  integrante dos estudos Alura - Challenger sobre analises de venda - DATA SCIENCE
Projeto desenvolvido como parte do curso **Praticando Python para Data Science: Challenge Alura Store** da Alura.  
O objetivo é explorar os dados de vendas da Alura Store e responder a perguntas de negócio usando Python e pandas.

## Objetivos da análise

- Calcular o faturamento total de cada loja.
- Identificar quais categorias de produtos são mais populares.
- Descobrir os produtos mais vendidos e menos vendidos.
- Analisar a média de avaliação dos clientes por loja.
- Calcular o valor médio de frete por loja.
- Verificar qual das lojas é a menos lucrativa com base no faturamento.

## Tecnologias utilizadas

- Python
- Jupyter Notebook
- pandas
- numpy
- matplotlib

## Estrutura do projeto

- `AluraStoreBrasil-4.ipynb`: notebook com todo o passo a passo da análise.
- `dados/`: pasta opcional para armazenar o arquivo CSV original de vendas.
- `README.md`: este arquivo, com a documentação do projeto.

## Como executar

1. Clone ou baixe este repositório.
2. Crie um ambiente virtual (opcional, mas recomendado).
3. Instale as dependências necessárias (por exemplo, `pandas` e `jupyter`).
4. Abra o arquivo `AluraStoreBrasil-4.ipynb` em um Jupyter Notebook ou JupyterLab.
5. Execute as células em ordem para reproduzir a análise.

## Principais insights

- Faturamento por loja, destacando qual tem maior e menor desempenho.
- Categorias com maior volume de vendas.
- Produtos que mais e menos vendem na base.
- Diferenças de avaliação média entre as lojas.
- Diferenças no valor médio de frete por loja.
